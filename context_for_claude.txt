1️⃣ Overall Objective

This project performs static and dynamic functional connectivity analysis on resting-state fMRI data to compare:

Healthy subjects

Parkinson’s Disease (PD) subjects

The goal is to investigate:

Differences in brain network topology

Differences in dynamic community reorganization

Differences in network flexibility (ROI switching behavior)

The project combines:

Neuroimaging preprocessing

Functional connectivity analysis

Spectral graph theory

Network science

Dynamic network analysis

2️⃣ Data Description

The project uses:

Resting-State fMRI Data

4D NIfTI files (.nii)

Dimensions: (X, Y, Z, Time)

Each voxel contains a BOLD time series

Atlas

AAL (Automated Anatomical Labeling) atlas

Used to parcellate brain into Regions of Interest (ROIs)

XML file provides ROI names

5️⃣ Core Modules and Their Roles
🔹 load_data.py
Responsibilities:

Load NIfTI fMRI images

Perform minimal preprocessing:

Detrending

Standardization

Band-pass filtering (0.01–0.1 Hz)

This isolates resting-state neural frequencies.

🔹 extract_bold.py

Uses:

NiftiLabelsMasker

Purpose:

Convert voxel-level data into ROI-level time series

Average signal within each atlas region

Output:

(time_points, number_of_selected_ROIs)

This reduces high-dimensional voxel space to region-level representation.

🔹 connectivity.py

Computes static functional connectivity.

Method:

Pearson correlation between ROI time series

Output:

(N × N) correlation matrix

Interpretation:

Nodes = ROIs

Edge weights = correlation strength

🔹 dynamic_analysis.py

This is the core nonlinear/dynamic component.

A. Sliding Window Connectivity

Implements:

sliding_window_connectivity(time_series, window_size, step_size)

Steps:

Divide BOLD time series into overlapping windows

Apply Gaussian tapering

Compute correlation matrix per window

Output:

(num_windows, N, N)

This captures time-varying connectivity.

B. Spectral Clustering

Uses graph Laplacian:

𝐿
=
𝐷
−
𝑊
L=D−W

Where:

W = adjacency (absolute connectivity)

D = degree matrix

Computes:

Eigenvalues

Fiedler vector (second smallest eigenvector)

Binary partition:

ROI assigned to cluster 0 or 1 based on Fiedler sign

Output:

(num_windows, N)

Each window has community labels.

C. ROI Switching Metric

Computes:

np.sum(np.abs(np.diff(cluster_matrix)), axis=0)

Measures:

How many times each ROI changes community across time

Interpretation:

Higher value → more dynamic

Lower value → more rigid network

🔹 network.py

Creates graph using NetworkX.

Steps:

Threshold connectivity matrix

Construct graph

Compute:

Degree centrality

Clustering coefficient

Top hubs

Used for static network comparison.

🔹 dynamic_visualization.py

Generates:

Animated connectivity heatmaps

Animated network graphs (cluster-colored)

ROI switching comparison bar plots

Outputs:

results/animations/
results/plots/
6️⃣ Static vs Dynamic Analysis
Static Analysis

Uses full time series correlation.

Produces:

Single connectivity matrix

Network graph

Hub analysis

Assumes connectivity is constant.

Dynamic Analysis

Uses sliding window approach.

Produces:

Time-varying connectivity matrices

Time-varying community structure

ROI flexibility metrics

Captures network reorganization over time.

7️⃣ Key Scientific Questions

This project investigates:

Do Parkinson’s subjects show altered static connectivity?

Do Parkinson’s subjects show reduced network flexibility?

Are specific ROIs more rigid in PD?

Does community structure reorganize differently in PD?

🧠 One-Sentence Summary

This project analyzes static and dynamic functional connectivity in resting-state fMRI data using spectral graph theory and sliding window methods to compare brain network flexibility between healthy and Parkinson’s subjects.
